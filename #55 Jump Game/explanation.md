I approached the problem by simply walking through the array and asking each index "is the number of jumps at this position greater than the jumps I currently have stored up from a previous position?". If the current position does have more jumps, i set my jumps variable to that and subtract 1 each iteration. I have a conditional checking if we are able to reach the end or if we get to a position that has value 0 and 0 jumps left, at which point we return false.
